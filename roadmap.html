07 · ADOPTION ROADMAP

From spreadsheets to supervised autonomy

A realistic 12-month path — each stage ships value on its own and de-risks the next.

Click a quarter to see its deliverables and exit criteria.

  1. Q1

    Codify the estate ▾

    Import existing infrastructure into Terraform; stand up the GitOps pipeline; asset register goes live from state. First win: audit evidence on demand.

    • Deliverables: Terraform modules for the top-20 systems · ArgoCD pipeline · auto-generated asset register
    • Process: PR template with change rationale · CODEOWNERS as approval matrix
    • Exit criteria: any auditor question about "who/what/when" answerable from Git in under a minute
  2. Q2

    Instrument & alert on SLOs ▾

    Observability stack with SLO-based alerting; incident tickets auto-created with context. First win: no more blank tickets at 3 a.m.

    • Deliverables: SLOs for top-10 services · Prometheus/Datadog alert rules · Slack + Jira integration
    • Process: alert review board retires noisy alerts monthly · every page must be actionable
    • Exit criteria: 90% of pages carry probable cause and a linked runbook
  3. Q3

    Close the FinOps loop ▾

    Idle reaping and off-hours scheduling in enforcement mode; weekly savings ledger to leadership. First win: measurable budget return.

    • Deliverables: idle-detection jobs · non-prod sleep schedules · anomaly alerts to resource owners
    • Process: savings ledger reviewed in the weekly leadership sync · owners accountable for unit costs
    • Exit criteria: realized savings visible as a budget line; forecast accuracy within ±5%
  4. Q4

    Introduce supervised agents ▾

    Triage and diagnostics agents in read-only mode → approval-gated remediation for proven patterns. First win: MTTR drops without headcount.

    • Deliverables: triage agent (L0→L1) · MCP tool servers with scoped credentials · agent action audit log
    • Process: autonomy ladder reviews — an agent is promoted to L2 only after N clean supervised runs
    • Exit criteria: MTTR down ≥50% on covered services; zero unauthorized agent actions
